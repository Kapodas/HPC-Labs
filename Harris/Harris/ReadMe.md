# Harris Corner Detection with CUDA
## Описание проекта
Этот проект реализует алгоритм обнаружения углов Харриса на GPU с использованием CUDA. Программа загружает изображение в формате BMP, 
выполняет обнаружение угловых точек с помощью CUDA, отмечает найденные угловые точки на изображении красным цветом и сохраняет результат в новый файл BMP.
