## Медианный фильтр для удаления шума "Соль и Перец" с использованием CUDA
### Описание проекта
Этот проект реализует медианный фильтр для удаления шума "Соль и Перец" из изображений с использованием CUDA. 
Медианный фильтр является эффективным методом для удаления такого типа шума, так как он заменяет значение каждого пикселя 
на медианное значение его соседей. Этот подход хорошо параллелизуется, что делает его идеальным для реализации на GPU с использованием CUDA.

Проект успешно реализует медианный фильтр для удаления шума "соль и перец" из изображений с использованием CUDA. Результаты показывают,
что использование GPU значительно ускоряет обработку изображений по сравнению с CPU. Проект также демонстрирует эффективное использование 
текстурной памяти в CUDA для улучшения производительности.
